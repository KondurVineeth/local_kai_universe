# Completions (Legacy)

Text completion for base models (legacy OpenAI endpoint).

> [!WARNING]
> This endpoint is no longer supported by OpenAI. ZL Universe continues to support it.
>
> Using this endpoint with chat-tuned models may produce unexpected tokens. Prefer base models.

- Method: `POST`
- Prompt template is not applied
- See OpenAI docs: [https://platform.openai.com/docs/api-reference/completions](https://platform.openai.com/docs/api-reference/completions)
